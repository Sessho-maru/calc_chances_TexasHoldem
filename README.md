# Texas Holdem possibility calculator
This program calculates your luck and possibility for each hand values on one Texas Holdem game,<br/>
based on cards in your hand and community<br/>

Calculates at every Betting `Pre-Flop`, `The Flop`, `The Turn`, `The River`<br/><br/>
